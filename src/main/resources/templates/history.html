Почему данные могут отличаться от реальной активности
  • Коммиты считаются по email в git-коммите. Если разработчик коммитил с другого адреса (личного, GitHub no-reply и т.д.) — эти коммиты не будут учтены. Проверьте git config user.email и добавьте нужный email как псевдоним в Настройки → Сотрудники.
  • MR привязываются по GitLab user ID. Если аккаунт GitLab не связан с сотрудником, его MR не попадут в статистику. Убедитесь, что при добавлении сотрудника был корректно указан GitLab username.
  • Аппрувы и комментарии также атрибутируются по GitLab user ID — та же причина.
  • История строится на еженедельных снимках. Если снимков за нужный период нет, запустите бэкфилл через Настройки.
